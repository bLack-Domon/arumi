Panduan Penggunaan Aplikasi Buku Induk Untuk Administrator

  • Menu Beranda

    Dalam Menu Beranda berisikan panduan singkat penggunaan aplikasi buku induk. Dengan fungsi-fungsi yang telah disesuaikan untuk kebutuhan pengolahan data santri di PP. Jalaluddin Ar-Rumi Jati Sari Jember. Serta informasi singkat mengenai jumlah santri putra, putri dan alumni.

  • Menu Data Santri

    Menu Data Santri menjadi pusat informasi yang dikelola dan ditampilkan dalam bentuk daftar tabel agar mudah untuk dibaca. Pengelola data dalam hal ini adalah Admin, pada menu ini admin juga dapat melakukan penyalinan data ke dalam bentuk excel untuk keperluan lainnya.

    1. Tombol Tambah: Submenu yang berada pada Menu Data Santri untuk melakukan pendaftaran santri baru.
    2. Tombol Detail: Submenu yang berada pada Menu Data Santri untuk melihat secara detail profil santri.
    3. Tombol Edit: Submenu yang berada pada Menu Data Santri untuk merubah atau mengupdate data santri.
    4. Tombol Berhenti: Submenu yang berada pada Menu Data Santri untuk merubah atau mengupdate status santri.
  • Menu Data Alumni

    Menu Data Alumni menjadi pusat informasi para alumni yang telah menyelesaikan pendidikan di PP. Jalaluddin Ar-Rumi.

    1. Tombol Detail: Submenu yang berada pada Menu Data Alumni untuk melihat detail profil santri.
    2. Tombol Update: Submenu yang berada pada Menu Data Alumni untuk merubah data apabila terjadi kesalahan status.
  • Menu Data Wilayah

    Menu Data Wilayah berfungsi sebagai pusat pengelolaan user (wilayah) serta daftar-daftar wilayah yang ada di PP. Jalaluddin Ar-Rumi

  • Menu Logout

    Menu untuk keluar dari sistem dan mengakhiri user yang sedang aktif.
