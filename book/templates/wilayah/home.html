Panduan Penggunaan Buku Induk

  • Menu Beranda

    Dalam Menu Beranda berisikan panduan singkat penggunaan aplikasi buku induk. Dengan fungsi-fungsi yang telah disesuaikan untuk kebutuhan pengolahan data santri di PP. Jalaluddin Ar-Rumi Jati Sari Jember.

  • Menu Data Santri

    Menu Data Santri menjadi pusat informasi yang dikelola dan ditampilkan dalam bentuk daftar tabel agar mudah untuk dibaca. Pengelola data dalam hal ini adalah Admin, pada menu ini admin juga dapat melakukan penyalinan data ke dalam bentuk excel untuk keperluan lainnya.

    1. Tambah Santri: Submenu yang berada pada Menu Data Santri untuk melakukan pendaftaran santri baru.
    2. Detail Santri: Submenu yang berada pada Menu Data Santri untuk melihat secara detail profil santri.
    3. Edit Santri: Submenu yang berada pada Menu Data Santri untuk merubah atau mengupdate data santri.
  • Menu Logout

    Menu untuk keluar dari sistem dan mengakhiri user yang sedang aktif.
